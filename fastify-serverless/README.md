# Getting Started with [Fastify-CLI](https://www.npmjs.com/package/fastify-cli)
This project was bootstrapped with Fastify-CLI.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

To start the app in dev mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm start`

For production mode

### `npm run test`

Run the test cases.

## WHAT DOES HE DO

The code sets up a Fastify server and defines an export function that can be used to handle HTTP requests in an asynchronous environment, ensuring that all Fastify songs and plugins are ready before handling requests.
